# CSShortcut App

> Projeto Descolado <3

## Stack

- Task Runner: [Gulp](http://gulpjs.com/)
- Html Template Engine: [Pug](https://pugjs.org/api/getting-started.html)
- Css Preprocessor: [Stylus](http://stylus-lang.com)
